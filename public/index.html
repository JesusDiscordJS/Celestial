Celestial Tracker

Bem-vindo ao Celestial User Tracker! Uma ferramenta avançada projetada para entusiastas do Discord e administradores de comunidade que desejam manter um registro histórico detalhado das atividades e perfis de usuários. Com o Celestial Tracker, você pode mergulhar fundo no passado digital de qualquer usuário monitorado pelo nosso bot Celestial.

Nossa plataforma permite visualizar todos os avatares que um usuário já utilizou, organizados cronologicamente, acompanhar as mudanças de nome de usuário ao longo do tempo, e obter informações sobre a atividade em chamadas de voz, como os últimos canais acessados. Além disso, tenha acesso a dados detalhados do perfil, incluindo o ID do usuário e estatísticas relevantes coletadas.

Para acessar a dashboard e todas essas funcionalidades, por favor, autentique-se usando sua conta do Discord. Sua privacidade e segurança são importantes para nós; utilizamos o sistema OAuth2 do Discord, que garante que apenas informações básicas de identificação sejam usadas para o login, sem acesso às suas mensagens ou dados sensíveis.
